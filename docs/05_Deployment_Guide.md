# Step 5: Deployment & Handover

## 1. Vercel 배포 설정
- `vercel.json` 설정 (프론트엔드와 백엔드 경로 설정).
- Vercel 대시보드에 환경 변수 등록:
  - `OPENAI_API_KEY`
  - `SUPABASE_URL`
  - `SUPABASE_ANON_KEY`

## 2. 배포 후 체크리스트
- Production URL에서 실제 분석 기능 작동 여부 확인.
- HTTPS 보안 연결 확인.

## 3. 최종 산출물 정리
- `README.md` 작성 (설치 및 실행 방법).
- API 명세 문서 정리.
